# Check-In CAE (Flutter)

## Ejecutar en desarrollo
```bash
flutter pub get
flutter run -d chrome
```

## Configurar servidor
En la pantalla de login puedes cambiar la `URL del servidor`.
- Web local: `http://localhost:4000`
- Android emulador: `http://10.0.2.2:4000`
- Dispositivo real: `http://IP_DE_TU_PC:4000`

## Permisos
La app solicita permisos de cámara y ubicación para el registro de ingreso/salida.
